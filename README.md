# Gurubharan Ganeson — academic website

This is a hand-coded static academic website designed for GitHub Pages.

## Files

- `index.html` — main site content
- `style.css` — visual styling
- `assets/profile.jpg` — optimized portrait image
- `assets/profile-square.jpg` — square crop, unused by default but useful for social previews or alternate layouts

## Before publishing

1. Replace the contact placeholder in `index.html` with your preferred email address.
2. Replace `https://bit.ly/44YtG7q` with a direct CV PDF path if you later want the CV hosted inside this repository.
3. Add PDF links to paper cards once drafts are available.

## Publish on GitHub Pages

1. Create a new GitHub repository, e.g. `gurubharan-website`.
2. Upload these files to the repository root.
3. In GitHub, go to `Settings` → `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select branch `main` and folder `/root`.
6. Save. GitHub will give you a public URL.

## Custom domain

If you buy a domain, add it under `Settings` → `Pages` → `Custom domain`.
GitHub will tell you the DNS records to add at your domain registrar.
